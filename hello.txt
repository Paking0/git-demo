git add name 添加到暂存区
git commit -m " first commit" name 提交到本地库 -m是提交信息
git reset --hard xxx  回退到xxx版本
branch 分支

git branch -v 参看分支
git branch 分支名  创建分支
切换分支：
git checkout 分支名
分支合并
git merge 要合并的分支  (会与当前分支合并)


产生冲突，合并分支时，两个分支在同一文件，同一位置有两套完全不同的修改。git无法决定使用哪个分支的内容，必须人为决定
修改后的提交 不能带文件名，才能合并

git push <remote别名> branch  提交到远程库
